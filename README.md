### 🔥 Featured Projects

<table>

<td width="50%">

**🌊 [FloatChatAI](https://github.com/SakethavasudevacharyaGundi/FloatChatAI)**

AI-powered ocean data explorer — query real ARGO float oceanographic data in natural language via a RAG pipeline (Gemini + ChromaDB), with an MCP server exposing 20+ scientific tools and interactive Plotly/Folium visualizations.

`Python` `FastAPI` `LangChain` `PostgreSQL` `RAG`

</td>
<td width="50%">

**⚙️ [Distributed Task Queue Engine](https://github.com/SakethavasudevacharyaGundi/Distributed_task_queue_engine)**

Production-grade distributed task queue in Go with priority scheduling, dead-letter queues, and watchdog crash recovery. Load-tested to **6,700+ req/sec** with a 0% failure rate and ~30ms p95 latency across 200,000+ submissions.

`Go` `Redis` `PostgreSQL` `Prometheus` `Grafana`

</td>
</tr>
<tr>
<td width="50%">

**💳 [Finsight](https://github.com/SakethavasudevacharyaGundi/Finsight)**

End-to-end analytics platform tracking India's UPI/digital payments ecosystem — automated RBI/NPCI ingestion through Airflow, modeled in Snowflake with dbt, visualized in Tableau. Processes 10+ years and 6,000+ records.

`Python` `Airflow` `Snowflake` `dbt` `Tableau`

</td>
<td width="50%">

**🔋 [FedGrid](https://github.com/SakethavasudevacharyaGundi/FedGrid)**

Privacy-preserving federated learning research on household energy forecasting (UK-DALE dataset) — compares local, federated, and centralized GRU models plus NILM appliance disaggregation, with federated learning coming out ahead on accuracy.

`Python` `PyTorch` `Federated Learning`
